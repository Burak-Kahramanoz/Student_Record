# Student Record Management System (C Language)

This is a simple console-based **Student Record Management System** written in C.  
It allows you to add, list, search, update, and delete student records using a singly linked list.

## Features

- Add new student records
- List all student records
- Search records by name or student number
- Calculate class average
- Delete a student record by number
- Show the most successful student (highest average)
- Update an existing student record

## Data Stored Per Student

Each student record contains:
- Student Number (int)
- Name (string)
- Midterm Grade (int)
- Final Grade (int)
- Average Grade (float)

## Menu Options

When you run the program, you'll be presented with a menu:
0 - Exit the program
1 - Add new student
2 - List/search student records
3 - Calculate class average
4 - Delete a student record
5 - Show the most successful student
6 - Update a student record
